# 布局

vue3-admin-template 封装了一套基础布局，放在了 `layout` 中，目录结构如下：

```sh
├── layout
│   ├── AppMain                   # 页面主内容
│   ├── GlobalBtn                 # 全局按钮
│   ├── Navbar                    # 导航栏
│   ├── PageFooter                # 页脚
│   ├── Sidebar                   # 侧边栏
│   ├── index.vue                 # layout 根组件
│   └── settings.js               # layout 配置
├── stores
│   └── app.js                    # layout 状态管理
└── styles
    └── layout.scss               # layout 样式
```
